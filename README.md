# ZSCodeUtil

## Author

Josh, 376019018@qq.com

## Desc

1. 二维码、条形码基础组件
2. 支持通过图片识别码
3. 扫描支持自动检测距离放大
4. 扫描支持自动检测亮度打开手电筒
5. 生成支持添加logo
6. 生成支持彩色

## Installation

TYPush is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'ZSCode'
```
